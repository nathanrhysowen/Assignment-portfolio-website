MY PORTFOLIO

About me

Hi, my name is Nathan Owen. I am a student studying Digital Media at University Of The West of England Bristol. Originally from Carlisle,
I moved down to Bristol to study my course. I have created this website as a portfolio to showcase the skills i have learned through my
course so far, ranging from knowledge of Adobe products like Indesign, illustrator and photoshop, to HTML and CSS programming.

Coding tasks

Task 1

The first task of the module introduced me to the very
basic fundamentals of HTML, including the body tag. It
also introduced me to Gitchub. Both of which i had no
prior knowledge of. Even though it was a very simple
task, it helped me understand the basic structure of
HTML programming.

Github link

Task 2

The second task of the module introduced me to CSS, which
allowed me to design and decorate the webpage with colour.
I also learned how to add images into a webpage. This is
where the programming began to get more fun and interesting,
as I enjoy the process of making the layout of the webpage
look nice. At this point I was still unsure about HTML and CSS
programming however it definitely gave me more confidence.

Github link

Task 3

The third task of the module expanded the CSS knowledge
we used in the previous task by making me link to a
seperate page within the first page in the previous task.
This task introduced concepts such as: selectors,styling
elements,border,padding and flexbox. All of which I have
been using within this portfolio website. I remember finding
this task slightly more challenging than the previous tasks
since it introduced so many new concepts and i was still
new to HTML and CSS however, I still found it fun
regardless.

Github link

Task 4

The third task of the module delved deeper into flexbox, which
was touched upon in the previous task. I learned how to style
my buttons using flexbox. I also learned how to turn my buttons
into real working buttons by placing an anchor link inside of them
using the anchor tag that sends the user to the specified link. I
found this task less challenging than the previous one ,as I was
more familiar with the way that CSS worked, however, I did have
trouble trying to position my flexbox buttons as they weren't going
where I wanted them.

Github link
